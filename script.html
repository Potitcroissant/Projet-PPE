Scripts Résultats

Objectif du scipt

Ces scripts Bash ont été créé pour automatiser la lecture, l'analyse et le traitement de fichiers contenant une liste d'environ 50 URLS chacun. Il a pour objectif la création d'un tableau HTML analysant un mot précis dans diverses langues. Dans le cas de nos 3 langues, on parlera de :

نظر | GAZE | REGARD

Présentation

Le script de ce projet est le même pour les 3 langues et varie seulement pour les chemins qui auront souvent ce type de début :

  • Arabe
  • Anglais
  • Français

Celui-ci nous a permis de créer la base de notre projet, il a été agencé de sorte à ce qu'un changement affecte tout ce à quoi il est relié. Il a 3 variantes qui souvent sont écrite ainsi : "script-langue.sh"

Fonctions principales

Récupération et analyse des données d'internet

En parcourant le.s fichier.s texte.s contenant les URLS, le script effectue d'abord pour chaque adresse :

  • Une vérification technique : il récupère le code de réponse HTTP de la page et identifie l'encodage des caractères qui seront pour la majorité en UTF-8 grâce aux formules suivantes :
  • Un nettoyage du contenu : il utilise ensuite l'outil lynx pour extraire uniquement du texte et retire donc les balises
  • Des statistiques textuelles : il compte automatiquement le nombre total de mots dans la page et le nombre d'occurences de notre mot-clé.
