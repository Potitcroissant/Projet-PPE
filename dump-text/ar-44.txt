    <span class="black" style="font-size:20px;"><h1 style="font-weight: bold;font-size: larger;margin-bottom:0 !important;border-bottom:none !important;">النظر</h1></span><span class="Tools-pdf" style="float:left;"><a href="#" disabled="disabled" title="سرویس pdf موقتا در دسترس نمی باشد">احفظ هذه المقالة بتنسيق PDF</a></span><br /><br/><hr /><br/>النّظر في إصطلاح <a href="https://ar.wikifeqh.ir/الفلاسفة" title="الفلاسفة" style="color:blue;" target="_blank" name="_الفلاسفة">الفلاسفة</a> عبارة عن ترتيب أمور <a href="https://ar.wikifeqh.ir/معلومة" title="معلومة (لا يوجد رابط)" style="color:red;" name="_معلومة">معلومة</a> متصوّرة، أو <a href="https://ar.wikifeqh.ir/مصدّق" title="مصدّق (لا يوجد رابط)" style="color:red;" name="_مصدّق">مصدّق</a> بها على وجه يؤدّى إلى <a href="https://ar.wikifeqh.ir/استعلام" title="استعلام (لا يوجد رابط)" style="color:red;" name="_استعلام">استعلام</a> ما ليس بمعلوم، فهو <a href="https://ar.wikifeqh.ir/حركة_النّفس" title="حركة النّفس (لا يوجد رابط)" style="color:red;" name="_حركة_النّفس">حركة النّفس</a> في <a href="https://ar.wikifeqh.ir/المعقولات" title="المعقولات (لا يوجد رابط)" style="color:red;" name="_المعقولات">المعقولات</a> مبتدئة من <a href="https://ar.wikifeqh.ir/المطلوب" title="المطلوب (لا يوجد رابط)" style="color:red;" name="_المطلوب">المطلوب</a> مستعرضة للمعاني <a href="https://ar.wikifeqh.ir/الحاضرة" title="الحاضرة (لا يوجد رابط)" style="color:red;" name="_الحاضرة">الحاضرة</a> عندها طالبة مباديه <a href="https://ar.wikifeqh.ir/المؤدّية" title="المؤدّية (لا يوجد رابط)" style="color:red;" name="_المؤدّية">المؤدّية</a> اليه.<br /><br /><div style="direction:rtl; </div> ترتيب أمور <a href="https://ar.wikifeqh.ir/معلومة" title="معلومة (لا يوجد رابط)" style="color:red;" name="_معلومة">معلومة</a> متصوّرة، أو مصدّق بها على وجه يؤدّى إلى استعلام ما ليس بمعلوم.<div class="tooltip"><a href="#foot2" name="foot-main2"> <sup>[۲]</sup></a>

</div> و <a href="https://ar.wikifeqh.ir/الفكر" title="الفكر" style="color:blue;" target="_blank" name="_الفكر">الفكر</a> بهذا <a href="https://ar.wikifeqh.ir/المعنى" title="المعنى" style="color:blue;" target="_blank" name="_المعنى">المعنى</a> (حركة <a href="https://ar.wikifeqh.ir/النّفس" title="النّفس" style="color:blue;" target="_blank" name="_النّفس">النّفس</a> في <a href="https://ar.wikifeqh.ir/المعقولات" title="المعقولات (لا يوجد رابط)" style="color:red;" name="_المعقولات">المعقولات</a> مبتدئة من <a href="https://ar.wikifeqh.ir/المطلوب" title="المطلوب (لا يوجد رابط)" style="color:red;" name="_المطلوب">المطلوب</a> مستعرضة للمعاني <a href="https://ar.wikifeqh.ir/الحاضرة" title="الحاضرة (لا يوجد رابط)" style="color:red;" name="_الحاضرة">الحاضرة</a> عندها طالبة مباديه المؤدّية اليه) هو الّذي يترتّب عليه <a href="https://ar.wikifeqh.ir/العلوم_الكسبيّة" title="العلوم الكسبيّة (لا يوجد رابط)" style="color:red;" name="_العلوم_الكسبيّة">العلوم الكسبيّة</a>، و ملاحظة <a href="https://ar.wikifeqh.ir/المعاني_الحاضرة" title="المعاني الحاضرة (لا يوجد رابط)" style="color:red;" name="_المعاني_الحاضرة">المعاني الحاضرة</a> عند الاستعراض على الوجه المذكور. و يسمّى أيضا نظرا.<div class="tooltip"><a href="#foot3" name="foot-main3"> <sup>[۳]</sup></a>
  <span class="tooltiptext">الإصفهاني شمس الدين، مطالع الأنظار، ص۱۰.</span>
</div> <a href="https://ar.wikifeqh.ir/اكتساب" title="اكتساب (لا يوجد رابط)" style="color:red;" name="_اكتساب">اكتساب</a> <a href="https://ar.wikifeqh.ir/المجهول" title="المجهول (لا يوجد رابط)" style="color:red;" name="_المجهول">المجهول</a> بالمعلومات <a href="https://ar.wikifeqh.ir/السّابقة" title="السّابقة (لا يوجد رابط)" style="color:red;" name="_السّابقة">السّابقة</a> على ذلك <a href="https://ar.wikifeqh.ir/المجهول" title="المجهول (لا يوجد رابط)" style="color:red;" name="_المجهول">المجهول</a>. ترتيب أمور معلومة أو <a href="https://ar.wikifeqh.ir/مظنونة" title="مظنونة (لا يوجد رابط)" style="color:red;" name="_مظنونة">مظنونة</a> للتأدّي إلى أمر آخر.<div class="tooltip"><a href="#foot4" name="foot-main4"> <sup>[۴]</sup></a>
  <span class="tooltiptext"><a href="lib.eshia.ir/71427/1/196/النظر" title="الايجي، السيد مير، شرح المواقف، ج1، ص196." name="outlink" target="_blank">الايجي، السيد مير، شرح المواقف، ج۱، ص۱۹۶.</a><span class="outlink">&nbsp;&nbsp;&nbsp;&nbsp;</span></span>

<script type="text/javascript">
function fixedEncodeURIComponent(str)
{
	return encodeURIComponent(str).replace(/[!'()]/g, escape).replace(/\*/g, "%2A");
}

function do_search(query)
{
	query = query.replace(/^\s+|\s+$/g, '');
	
	if (query && query != "جستجو...")
	{
		query = query.replace(/ +/g, '_').replace(/['\0\\]+/g, '');
		window.location.assign('/search/' + '?' + '&q=' + fixedEncodeURIComponent(query));
	}
	
	return false;
}
</script>
