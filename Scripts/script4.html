Documentation des Scripts

Automatisation, Analyse et Traitement de Corpus

Objectif du script

Ces scripts Bash ont été créés pour automatiser la lecture, l'analyse et le traitement de fichiers contenant une liste d'environ 50 URLS chacun. Ils ont pour but la création d'un tableau de bord HTML analysant un mot précis dans diverses langues.

Dans le cas de nos 3 langues, on parlera de :

نظر | GAZE | REGARD

Présentation du script

Les scripts de ce projet sont les mêmes pour les 3 langues et varient seulement pour les chemins qui auront souvent ce type de début : *ar, *en, *fr.Ils nous ont permis de créer la base de notre projet. Ils ont été agencés de sorte à ce qu'un changement affecte tout ce à quoi ils sont reliés.

Nos scripts ont pour nom : "script-"langue".sh".

Récupération et Analyse

En parcourant le fichier texte contenant les URLS, le script effectue:

  • Vérification technique : Récupération du code HTTP et identification de l'encodage (majoritairement UTF-8).
  • Nettoyage : Utilisation de lynx pour extraire le texte brut sans balises HTML.
  • Statistiques : Comptage automatique du nombre total de mots et d'occurrences du mot-clé.

Ressources linguistiques

Pour chaque URL, le script génère:

  • Aspirations : Copie locale du code source HTML.
  • Dump-texte : Version texte brut de la page.
  • Contextes : Extrait du mot-clé avec 4 lignes environnantes pour analyse sémantiqu.
  • Concordanciers : Tableaux HTML "Contexte gauche | Cible | Contexte droit".

Publication des Résultats

Le script agence les données pour un rendu visuel optimal:

  • Génération d'une page internet complète avec le framework Bulma.
  • Intégration d'un tableau récapitulatif avec liens directs vers les fichiers générés.
  • Menu de navigation pour basculer entre les langues et thématiques.
Utilisation sur le terminal

Pour tester le fonctionnement sans générer la page HTML finale:

  • Arabe : bash programmes/scripts-fr.sh URLs/ar.txt
  • Anglais : bash programmes/scripts-fr.sh URLs/en.txt
  • Français : bash programmes/scripts-fr.sh URLs/fr.txt

Commandes et Formules

Commande Explication
while read -r line Boucle pour lire le fichier d'URLS ligne par ligne. La variable $line stocke l'adresse.
curl -I Récupère les en-têtes du serveur (métadonnées).
grep -Po "charset=\S+" | cut -d"=" -f2 Isole le type d'encodage (ex: UTF-8).
lynx -dump -nolist Convertit le HTML en texte brut sans liens pour un comptage fiable via wc -w.
egrep -i -C 4 Extrait le mot cible avec un contexte de 4 lignes avant/après.
sed 's/(.*)($mot)(.*)/.../' Utilise des expressions régulières pour diviser le contexte et centrer le mot cible.
{...} > "tableaux.langue.html" Redirige les sorties echo directement dans le fichier HTML.
