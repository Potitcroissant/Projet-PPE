Pour pouvoir analyser les contextes dans lesquels le mot "gaze" est utilisé, nous faisons appel au script PALS.

Tout d’abord, nous devons tokéniser le corpus de texte afin que chaque ligne ne soit composée que d’un token. Pour cela,j’utilise les lignes de code suivante:

mot="[Gg]aze"
while read -r line
do
cat "dump-text/en-${lineno}.txt" | sed -E 's/([^[:space:][:punct:]]+|[[:punct:]])/\1\n/g' | grep -v '^[[:space:]]*$' > "corpus-texte/en/en-$lineno.txt"
lineno=$(expr $lineno + 1)
done < $fichier_urls

J’ai utilisé regex pour prendre en compte le mot avec ou sans majuscule.

Ensuite, nous utilisons le script PALS:

python3 ../PALS-main/cooccurrents.py --target [Gg]aze --match-mode regex corpus-texte/en/* > resultat.csv

J’ai exporté les résultats dans un fichier CSV pour plus de clarté.

Les analyses de cooccurrences montrent que les mots les plus utilisés avec "gaze" sont des mots ayant une connotation "sociale" comme par exemple, "male", "objectifying", "theory", "relationship", "power"...

On remarque aussi des mots plus "factuels" comme "look", "reading", "book", "image"...

De manière générale, ce mot est très utilisé en anglais dans le contexte de “male gaze”

Quant aux nuages de mots, j’ai concaténé le corpus dump pour avoir tous les mots dans un seul fichier et j’ai répété l’opération pour le corpus de contextes. Ensuite j’ai utilisé les commandes suivantes pour créer un nuage de mots:

wordcloud_cli --text nuages/dump-en.txt --imagefile nuages/dump.png
wordcloud_cli --text nuages/contextes-en.txt --imagefile nuages/contexte.png
